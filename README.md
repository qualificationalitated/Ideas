- ## 앵겔지수 + 술

  앵겔지수는 수입에서 식료품이 차지하는 비율

  데이터분석 쪽으로 재미있는걸 해볼수있지 않을까

  각자 직업별로 판단하여, 넣어볼수도 있고,

  술은 스트레스를 해소하기위해 마신다 라는 가설을 세운 뒤, 개인의 스트레스 통계와 비교분석을 통해 유의미한 결과값을 도출해낼수 있을것이라 생각한다.

  주류회사의 마케팅에도 활용가능 하지 않을까 생각한다.
